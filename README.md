1. Менеджер задач с умными напоминаниями (Advanced Тodo)
Суть: Бот для планирования задач, но не просто список, а с гибкими уведомлениями.

Что под капотом (Стек): SQLite / PostgreSQL, APScheduler (для отправки уведомлений в точное время), State Machine (FSM).

Что оценивает работодатель: Умение работать с состояниями пользователя (пошаговый ввод), правильную работу с датами/часовыми поясами и фоновыми задачами.

2. Мультивалютный трекер цен (Крипта / Фиат)
Суть: Бот парсит актуальные курсы валют или криптовалют, позволяет настраивать алерты (например: «уведоми, если Bitcoin упадет ниже $60k»).

Что под капотом (Стек): Внешние API (CoinGecko / ExchangeRates), кэширование (чтобы не спамить API при каждом запросе), библиотека для графиков (matplotlib / quickchart.io) для вывода красивой инфографики изменений курса.

Что оценивает работодатель: Работа со сторонними REST API, обработка ошибок сети, базовое кэширование данных.# -
